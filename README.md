# 📚 Online Book Store Sales Analysis

## 🔍 Overview

This project focuses on analyzing sales data from an online bookstore using SQL and visualizing key insights through an interactive Power BI dashboard. The goal is to extract meaningful business insights such as revenue trends, customer behavior, and product performance.


## 🗂️ Dataset

The dataset consists of three main tables:

* **Books** – Contains book details (title, author, genre, price, stock)
* **Customers** – Stores customer information (name, location, contact)
* **Orders** – Includes transaction data (order date, quantity, total amount)


## 🛠️ Tools Used

* **SQL Server Management Studio (SSMS)** – Data storage and querying
* **SQL** – Data extraction and transformation
* **Power BI** – Data visualization and dashboard creation


## ⚙️ Steps

* Created and structured the database in SQL Server
* Imported datasets into respective tables
* Performed data cleaning and validation
* Wrote SQL queries (basic to advanced) to analyze:

  * Sales performance
  * Customer behavior
  * Inventory insights
* Connected SQL data to Power BI
* Built an interactive dashboard with filters and visuals


## 📊 Dashboard

The Power BI dashboard includes:

* KPI cards (Total Revenue, Orders, Customers, Books Sold)
* Sales distribution by genre
* Monthly order trends
* Top-selling books
* Author-wise performance
* Interactive filters (Genre, Author, Country)


## 📈 Results / Insights

* Identified top-performing genres and authors
* Analyzed monthly sales trends and order patterns
* Determined high-value customers and spending behavior
* Evaluated stock levels and inventory efficiency


## ▶️ How to Run

* Open SQL Server Management Studio
* Execute the provided SQL script to create the database and tables
* Import the dataset into the tables
* Run the SQL queries to verify data
* Open Power BI Desktop
* Connect to the SQL Server database
* Load data and open the dashboard file
